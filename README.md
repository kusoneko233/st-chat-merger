# 🔀 Chat Merger - 聊天记录合并工具

**版本** `1.0.0` **兼容** SillyTavern

> SillyTavern 扩展插件 — 多聊天合并 + 智能去重 + 预览导出

---

## 🚀 安装方法

### 方法一：通过 URL 安装（推荐）

1. 打开 SillyTavern → 点击顶部的 **扩展 (Extensions)** 图标
2. 点击 **安装扩展 (Install Extension)**
3. 在输入框粘贴仓库地址：
https://github.com/kusoneko233/st-chat-merger
4. 点击安装，等待完成后刷新页面即可

### 方法二：手动安装

1. 下载本仓库 ZIP 包（Code → Download ZIP）
2. 解压后，将文件放入：
SillyTavern/public/scripts/extensions/third-party/st-chat-merger/
3. 确保目录结构如下：
st-chat-merger/
   index.js
   merger.html
   manifest.json
   style.css
4. 重启 SillyTavern

---

## 🛠️ 使用说明

1. **打开插件**：点击酒馆右侧扩展面板（魔法棒图标），找到「合并聊天记录」并点击
2. **选择聊天**：
- **当前角色卡聊天**标签页：勾选要合并的聊天记录
- **手动导入外部文件**标签页：拖入或选择外部 JSONL 文件
3. **设置选项**：
- 选择合并排序方式
- 建议勾选「仅保留第一个聊天的首条消息」
- 按需选择去重、分隔标记等选项
4. **预览检查**：点击「生成预览」查看合并结果，确认无误
5. **执行合并**：点击「合并并导入」直接导入酒馆，或点击「合并并下载」保存文件

## 📖 简介

当你的角色卡更新换代、或者聊天记录分散在多个文件中时，它能帮你把所有对话整合为一个连续的聊天文件。支持自动读取当前角色卡聊天、手动导入外部文件、智能去重、合并预览等功能，让聊天记录管理变得简单高效。

---

## ✨ 核心功能

### 1. 📋 当前角色卡聊天合并
- **一键读取**：自动获取当前角色卡的所有聊天记录列表，按时间从早到晚排列
- **灵活选择**：支持全选/全不选，勾选需要合并的聊天
- **手动排序**：通过上下箭头调整聊天的合并顺序
- **详细信息**：每个聊天显示消息条数、文件大小、创建时间，当前聊天会高亮标记

### 2. 📂 外部文件导入
- **拖拽导入**：直接拖拽 JSONL 文件到导入区域，或点击选择文件，支持多选
- **跨角色卡合并**：适用于角色卡版本更新后，将旧角色卡的聊天迁移到新角色卡
- **自由调序**：导入后可上下移动调整顺序，也可单独移除某个文件

### 3. ⚙️ 合并选项
- **合并排序**：按列表顺序（时间从早到晚）/ 按手动调整的序号 / 按文件名升序
- **首条消息去重**：仅保留第一个聊天的首条系统消息，避免合并后出现多条重复的角色卡信息（强烈建议勾选）
- **消息去重**：移除发送者+内容+时间完全相同的重复消息（适用于文件间有重叠内容的场景）
- **分隔标记**：在合并衔接处插入 `───── ✂ 以下合并自: 文件名 ─────` 的分隔消息，方便定位交界
- **自定义文件名**：可指定输出文件名，留空则自动生成

### 4. 👁️ 合并预览
- **三段式预览**：智能显示开头、拼接处、结尾的关键消息，省略中间大段内容
- **全屏查看**：支持全屏展开查看完整合并结果
- **统计面板**：实时显示文件数、总消息数、去重数、最终消息数

### 5. 📤 导出方式
- **合并并导入**：一键合并后直接通过酒馆的导入聊天功能导入，无需手动操作
- **合并并下载**：合并后下载 JSONL 文件到本地，可自行手动导入
